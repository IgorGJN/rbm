CRIADOR DE INFORMATIVOS RBM - VERSÃO COM BLOCO PERSONALIZADO

Como abrir:
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Rode com Live Server e abra o index.html.

Arquivos principais:
- index.html
- styles.css
- blocks.js
- templates.js
- storage.js
- renderer.js
- app.js

Novidades desta versão:
- Mantida a base atual do editor, sem voltar para versões anteriores.
- Inclusão de controle de cor de fonte em vários blocos:
  - capa;
  - título de seção;
  - texto;
  - destaque;
  - bloco institucional;
  - colunas numeradas;
  - checklist;
  - legenda de imagem;
  - botões.
- Novo bloco chamado "Bloco personalizado".
- O Bloco personalizado permite:
  - largura 100%, 75%, 2/3, 50%, 1/3 ou 1/4;
  - criar blocos lado a lado, como dois blocos de 50% ou três blocos de 1/3;
  - título opcional;
  - texto opcional;
  - lista de pontos opcional;
  - imagem opcional;
  - botão opcional;
  - cor de fundo própria;
  - cor de título própria;
  - cor de texto própria;
  - cor da borda;
  - cor de detalhe;
  - cor do botão;
  - alinhamento do texto;
  - imagem acima, à esquerda ou à direita;
  - borda normal, borda lateral ou sem borda;
  - espaçamento interno;
  - arredondamento;
  - sombra leve.

Como usar blocos lado a lado:
1. Clique em "Blocos".
2. Adicione "Bloco personalizado".
3. No painel lateral, altere a largura para 50%, 1/3 ou 1/4.
4. Adicione outro "Bloco personalizado" logo em seguida e escolha largura compatível.

Exemplos:
- 2 blocos lado a lado: use 50% + 50%.
- 3 blocos lado a lado: use 1/3 + 1/3 + 1/3.
- 4 blocos lado a lado: use 1/4 + 1/4 + 1/4 + 1/4.

Observações:
- Para editar o fundo da página, clique em uma área livre da página ou no botão "Página" no canto superior direito da folha.
- Para editar um bloco, clique diretamente no bloco.
- Para PDF, use o botão "Imprimir / PDF" e salve como PDF no navegador.
- Se aparecer alerta de corte, mova conteúdo para outra página em vez de reduzir demais as fontes.
- O novo bloco personalizado respeita a impressão A4, mas blocos lado a lado precisam de textos mais curtos para evitar corte.

VERSÃO V6 - AJUSTES DE USABILIDADE

Melhorias incluídas:
- Ajustes rápidos por bloco no painel direito:
  - espaço acima;
  - espaço abaixo;
  - tamanho do título;
  - tamanho do texto;
  - altura da linha.
- Blocos personalizados lado a lado agora podem ficar com altura igual automaticamente.
- O bloco personalizado ganhou painel mais compacto, dividido em:
  - Layout;
  - Conteúdo;
  - Fonte;
  - Visual.
- Os controles principais do bloco personalizado foram transformados em botões lado a lado para reduzir a rolagem.
- O espaçamento padrão entre blocos foi suavizado para evitar quadros muito distantes.

Dica de uso:
- Para cards lado a lado, use o Bloco personalizado com largura 50%, 1/3 ou 1/4.
- Para aproximar ou afastar blocos, selecione o bloco e use Espaço acima / Espaço abaixo.
- Para equilibrar cards lado a lado, deixe marcada a opção Igualar altura quando estiver lado a lado.
