CRIADOR DE INFORMATIVOS RBM - VERSÃO AJUSTADA

Como abrir:
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Rode com Live Server e abra o index.html.

Principais correções desta versão:
- Opacidade real das imagens: 100% deixa a imagem totalmente visível, sem camada branca/azul por cima; 0% esconde a imagem.
- Imagem de fundo da página corrigida com controle direto de opacidade.
- Imagem de fundo da capa corrigida com controle direto de opacidade.
- Fundo de página com duas opções: degradê suave ou cor sólida completa.
- Blocos com cor de fundo editável, e não apenas cor de detalhe.
- Capa sem azul fixo. Agora a capa pode usar branco, cinza claro, dourado suave, dourado forte, azul navy, azul institucional, azul médio, verde, vermelho ou cinza.
- Destaques, bloco institucional, colunas e checklist agora têm escolha de cor de fundo.
- Botões agora permitem escolher a cor principal.
- Modelos iniciais ajustados para visual mais leve, com capa em dourado suave por padrão.

Observações:
- Para editar o fundo da página, clique em uma área livre da página ou no botão "Página" no canto superior direito da folha.
- Para editar um bloco, clique diretamente no bloco.
- Para PDF, use o botão "Imprimir / PDF" e salve como PDF no navegador.
- Se aparecer alerta de corte, mova conteúdo para outra página em vez de reduzir demais as fontes.
